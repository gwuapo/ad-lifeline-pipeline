# Static Ad Generator SOP

## Purpose
Create high-converting static ad concepts with specific design direction, copy, and layout.

## Process

### Step 1: Creative Brief
- Product and offer
- Platform and placement (Meta feed 1:1, Story 9:16, TikTok, etc.)
- Target avatar
- Primary message / angle
- Competitive landscape (what ads are they already seeing?)

### Step 2: Concept Generation
Generate 3-4 distinct concepts. For each:

**Layout & Composition:**
- Visual hierarchy (what does the eye see first, second, third?)
- Image/graphic placement
- Text placement and sizing
- Whitespace usage
- Overall style (clean minimal, bold graphic, UGC-style, editorial, meme-format)

**Copy Elements:**
- Headline (max 8 words, benefit-driven)
- Subheadline (supports the headline, handles objection)
- Body text if needed (keep minimal on static)
- CTA button text

**Visual Direction:**
- Photography style or illustration approach
- Color palette (hex codes)
- Typography (font style, weight, size hierarchy)
- Any icons, badges, or trust elements
- Before/after, comparison, or data visualization if applicable

### Step 3: Platform Optimization
- Recommended dimensions for each platform
- Safe zones for text (Meta 20% text rule considerations)
- Mobile-first readability check
- Thumb-stopping potential score (1-10)

### Step 4: A/B Test Variants
For the top concept, create 2-3 variations:
- Different headline
- Different visual treatment
- Different CTA

## Quality Standards
- Must be thumb-stopping -- would YOU stop scrolling?
- Text must be readable at mobile size
- One clear message per ad (don't try to say everything)
- Design specs must be detailed enough for a designer to execute
- Reference swipe file for proven static ad formats
