# Ad Copy Writer SOP

## Purpose
Generate high-converting direct response ad copy and video scripts that drive action.

## Process

### Step 1: Brief Analysis
- Product, offer, and unique mechanism
- Target avatar and their awareness level (unaware → most aware)
- Platform and format (Meta feed, TikTok, YouTube pre-roll, etc.)
- Objective (purchase, lead, webinar registration, etc.)
- Any constraints (compliance, claims we can't make)

### Step 2: Hook Generation
Generate 5-8 hooks using these proven patterns:
- **Curiosity gap**: "I discovered something about [topic] that changed everything"
- **Contrarian**: Challenge a common belief
- **Pain callout**: Name the exact frustration
- **Result lead**: Start with the transformation
- **Story open**: "3 months ago I was [relatable situation]"
- **Pattern interrupt**: Unexpected visual or statement
- **Authority**: Leverage credible source

### Step 3: Script / Copy Structure
For VIDEO scripts:
```
[HOOK - 0:00-0:03] Scroll-stopper. Must create curiosity or emotion.
[PROBLEM - 0:03-0:15] Agitate the pain. Make them feel understood.
[FAILED SOLUTIONS - 0:15-0:25] "You've tried X, Y, Z and nothing worked"
[MECHANISM - 0:25-0:40] Introduce the unique mechanism (not the product yet)
[PRODUCT REVEAL - 0:40-0:50] Now introduce the product as the vehicle
[PROOF - 0:50-1:10] Social proof, results, testimonials, data
[CTA - 1:10-1:20] Clear, urgent call to action with reason to act now
```

For STATIC/TEXT copy:
```
HEADLINE: Benefit-driven, specific
SUBHEAD: Expand on the promise or handle #1 objection
BODY: PAS or AIDA framework, 3-5 short paragraphs
CTA: Action verb + benefit + urgency
```

### Step 4: CTA Variations
Generate 5 CTAs with different urgency/scarcity angles

### Step 5: Testing Notes
- What to A/B test first (hook vs hook)
- What to iterate on if performance is flat
- Platform-specific optimization notes

## Quality Standards
- Every sentence must earn attention or drive action -- no filler
- Write at a 6th grade reading level
- Use conversational, natural language (not "marketing speak")
- Reference the swipe file for proven patterns
- Include stage directions for video [SHOW product, CUT TO testimonial, etc.]
