# Saudi Najdi Arabic Dialect Guide for Ad Translation

## Common Word Substitutions (MSA → Najdi)

| English | MSA (WRONG) | Najdi (CORRECT) |
|---------|-------------|-----------------|
| What | ماذا | ايش / وش |
| Want | يريد | يبي / يبغى |
| Like this | هكذا | كذا |
| Now | الآن | الحين |
| A lot | كثيراً | وايد / كثير |
| Why | لماذا | ليش |
| How | كيف | كيف (same) |
| Where | أين | وين |
| Good | جيد | زين / حلو |
| Bad | سيء | خايس |
| Come | تعال | تعال (same) |
| Go | اذهب | روح |
| See/Look | انظر | شوف |
| Give | أعطِ | عطني / اعطني |
| Know | يعرف | يدري |
| Speak/Talk | يتحدث | يتكلم |
| Thing | شيء | شي |
| Person | شخص | واحد |
| Very | جداً | مرة / وايد |
| Finished | انتهى | خلص |

## Sentence Pattern Rules

1. Questions often start with "وش" or "ايش" instead of standard question particles
2. Negation uses "ما" before the verb: "ما ابي" (I don't want)
3. Future tense uses "بـ" prefix: "بروح" (I will go)
4. Possessive uses "حق": "السيارة حقي" (my car) instead of "سيارتي"

## Ad Copy Style Notes

- Keep sentences SHORT — Najdi speakers don't use long complex sentences in casual speech
- Use direct address: "انت" not "أنتم" for second person (unless genuinely plural)
- Brand names, product names, numbers stay in English/digits
- Urgency words: "الحين" (now), "اليوم" (today), "لا تفوت" (don't miss)
